## 🌐 إضافة مزود API جديد (مثل G2Bulk) — خطوة بخطوة

### 1️⃣ إضافة المزود
1. افتح تطبيق الأدمن ← القسم **"مزودو API"**.
2. اضغط **"إضافة مزود"**.
3. املأ الحقول:
   - **الاسم**: اسم المزود (مثلاً: `SMS Hub`).
   - **رابط API الأساسي**: الرابط الكامل بدون نقطة النهاية (مثلاً: `https://api.smshub.com/v1/`).
   - **مفتاح API**: المفتاح الذي حصلت عليه من حسابك عند المزود.
   - **اسم الهيدر**: عادة `X-API-Key` (أو `Authorization` لبعض المزودين).
   - **نوع التحقق**: `header` (الأكثر شيوعاً).
4. اضغط **حفظ**.

### 2️⃣ ضبط هامش الربح
- في حقل **"هامش الربح %"** ضع النسبة المئوية التي تُضاف لسعر المزود.
- مثال: `16` تعني منتجاً بسعر `1$` سيُباع للمستخدم بـ `1.16$`.
- النسبة تُطبَّق تلقائياً على كل المنتجات.

### 3️⃣ مزامنة المنتجات
- اضغط زر **"مزامنة"** بجانب المزود.
- سيتم جلب:
  - **التصنيفات** (categories)
  - **المنتجات** (products)
  - **الألعاب** (games — لـ G2Bulk)
- قد تستغرق المزامنة 1-3 دقائق للمزودين الكبار.

### 4️⃣ إنشاء قسم رئيسي للمزود (اختياري)
1. اذهب لقسم **"إدارة الأقسام"**.
2. اضغط **"إضافة قسم"**.
3. املأ: الاسم، الأيقونة، نوع القسم = **API**.
4. اختر المزود من القائمة المنسدلة.
5. اضغط **حفظ** — ستظهر منتجات المزود تلقائياً داخل هذا القسم في تطبيق المستخدم.

### 5️⃣ إنشاء قسم فرعي (اختياري)
1. افتح القسم الرئيسي الذي أنشأته.
2. اضغط **"إضافة قسم فرعي"**.
3. مثال: داخل قسم "الترفيه" يمكنك إنشاء: "ببجي"، "فري فاير"، "كول أوف ديوتي".
4. اربط كل قسم فرعي بـ ID التصنيف المناسب من المزود.

### 6️⃣ مراقبة رصيد المزود
- في صفحة المزود، اضغط أيقونة 🔄 لجلب الرصيد الحالي من `/getMe`.
- يظهر تنبيه أحمر إذا انخفض الرصيد عن `50$`.
- كل تحديث للرصيد يُسجَّل في قسم **"سجل الرصيد"**.
